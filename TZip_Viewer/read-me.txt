Installation

1. Copy TZip.pas and TZip.dcr into a directory.
2. In Delphi, select Component - Install menu option.
3. Select Add... button
4. Select Browse
5. Select TZip.pas in the directory from step 1
6. Select OK
   
The component library will now re-compile, and when it is finished you should
have a TZip component on your Samples tab.
